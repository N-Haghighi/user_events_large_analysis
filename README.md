# User Behavioral Analysis  
### Messaging Application Event Data (Python | Pandas)

---

## Overview

This project presents an exploratory behavioral analysis of event-level interaction data from a simulated messaging application.

The objective was to transform raw user activity logs into structured engagement insights using a reproducible Python-based data processing workflow.

---

## Dataset Summary

- 1,500 total events  
- 100 unique users  
- 15-day observation window  
- Event types: `login`, `message_sent`  
- Platforms: Android, iOS  
- Multi-city distribution  

---

## Analytical Focus

The analysis covers:

- Engagement concentration  
- Active user segmentation  
- Temporal usage patterns  
- Platform distribution  
- Retention evaluation  
- KPI-driven behavioral insights
